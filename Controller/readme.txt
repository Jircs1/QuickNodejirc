Controllers are here

Use Domain/Controller/Action to request APIs.

Controller are the class files in this directory, Action is the static function

Keep the name of the class file the same with the class name inside the file.

For example, when you create a file called Index.js. Please define the class named 'class Index'

控制器与方法，路由与ThinkPHP传统URL模式相同

使用 地址+控制器+方法名的方式访问API，Controller就是本目录中的类文件，Action是类下面的的方法

为了规范化，请保持JS类文件名和代码的类名相同，例如Index.js里应该是class Index
